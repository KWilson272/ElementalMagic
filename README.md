# ElementalMagic
ElementalMagic is a modernization of the PVP ability plugin utilized on the
minecraft server ``legacybending.us.to``, which itself is a rewrite of 
ProjectKorra 1.8.0-BETA-9.

The abilities in this plugin feature bug fixes, functionality tweaks, and 
additions that are specifically tailored for competitive PVP-based gameplay.
Survival support will be provided at a later time, but not prioritized. 
First and foremost, this plugin will be intended for use on LegacyBending; 
however, anyone is welcome to use it if it fits their needs.

**This port is currently in-progress and missing key features that make it 
inappropriate for use in a server environment.*

## Installation
- ``Java Version`` 21
- ``Spigot Version`` 1.21.11

Run ``./gradlew build`` in the root directory to build this plugin.

## Key Missing Features
The following important features are missing, but are planned for addition.
- Collision System
- Commands
- Region Protection
- Survival Integration
